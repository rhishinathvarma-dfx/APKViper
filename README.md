DarkAudit v2.0.0
Advanced Android Security Assessment Platform

A portable, offline-capable Android SAST tool with taint analysis, exploit intelligence, and enterprise-grade reporting.

Features
11 Analysis Engines — Taint tracking, call graph, cross-reference, string deobfuscation, manifest, code, network, binary, file, tracker, network config parser
50+ Security Rules — OWASP MASVS, CWE, CVSS 3.1 scored
Exploit Intelligence — Real-world PoCs with Frida/drozer/ADB scripts per finding
Bypass Techniques — Auto-generated bypass code for detected security controls
7 Report Formats — HTML, PDF, Excel, JSON, Word, PowerPoint, SARIF
3 Modes — GUI, CLI (headless), REST API server
Custom Rules — YAML-based rule authoring
Zero Install — Single portable EXE, fully offline
Quick Start
# GUI Mode (double-click)
DarkAudit.exe

# CLI Headless Scan
java -jar DarkAudit.jar --scan app.apk --format sarif --output report.sarif.json

# REST API Server
java -jar DarkAudit.jar --server --port 8089
Build
# Requires JDK 11+
./build-installer.bat
CI/CD Integration
# GitHub Actions
- run: java -jar DarkAudit.jar --scan app.apk --format sarif --output results.sarif.json
- uses: github/codeql-action/upload-sarif@v2
  with:
    sarif_file: results.sarif.json
Custom Rules
Place .yaml files in ~/.darkaudit/rules/:

id: CUSTOM-001
name: Hardcoded API Endpoint
severity: HIGH
pattern: "https?://target+/api/"
targets: [JAVA_SOURCE]
impact: Hardcoded endpoints can be extracted by attackers.
fix: Use BuildConfig or encrypted config.
cvss: 5.3
